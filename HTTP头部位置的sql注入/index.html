Rj45's blog
2019-09-24

前言

在前面，我们根据基于从服务器接收到的响应的方式将sql注入分为联合注入、报错注入、布尔盲注、时间盲注等。
当我们根据基于注入点的位置的时候，可以将sql注入分为通过用户输入的表单域的注入、通过cookie的注入和通过服务器变量注入(头部信息注入)也即HTTP头部注入。
