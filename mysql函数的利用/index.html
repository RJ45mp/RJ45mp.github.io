mysql字符串函数:

函数 作用
ascii(s) 返回字符串s的第一个字符的ascii码

flnd_in_set(s1,s2) 返回在字符串s2中与s1匹配的字符串的位置
format(x,n) 将数字x格式化，保留小数点后n位，最后一位四舍五入
insert(s1,x,len,s2) s2替换s1的x位置开始长度为len的字符串
locate(s1,s) 从字符串s中获取s1的开始位置
lcase(s) 将字符串s的所有字母变为小写字母
left(s,n) 返回字符串s的前n个字符
lower(s) 将字符串s的所有字母变为小写字母
lpad(s1,len,s2) 在字符串s1的开始处填充字符串s2,使字符串长度达到len
ltrim(s) 去掉字符串s开始处的空格
mid(s,n,len)\substr(s,n,len)\substring(s,n,len) 从s中的n位置开始截取长度为len的子字符串
position(s1 in s) 从字符串s中获取s1的开始位置
repeat(s,n) 将字符串s重复n次
replace(s,s1,s2) 将字符串s2替代串s中的字符串s1
reverse(s) 将字符串s的顺序反过来
right(s,n) 返回字符串s的后n个字符
rpad(s1,len,s2) 在字符串s1的结尾处添加字符串s2,使字符串的长度达到len
rtrim(s) 去掉字符串s结尾处的空格
space(n) 返回n个空格
strcmp(s1,s2) 比较s1和s2，如相同返回0，s1>s2返回1，否则返回-1
substring_index(s,delimiter,number) 返回从字符串s的第number个出现的分隔符delimiter之后的字串。如number是正数返回第number个字符左边的字符串。如number是负数返回第number的绝对值个字符右边的字符串。
trim(s) 去掉字符串s开始和结尾处的空格
ucase(s)\upper(s) 将字符串转换为大写

函数在注入过程中的利用

1、ascii(s)：返回字符串s的第一个字符的ascii码

可以用于布尔盲注中，结合ord()函数会更加直观显示。时间盲注也行

2、char_length(s)\character_length(s)：返回字符串s的字符数

在盲注中可以用到

3、concat(s1,s2,…,sn):合并s1,s2,…sn字符为一个字符

在注入，将多个字段的数据一列一次性返回，与concat_ws(0x7e,s1,s2,…,sn)一样。

不过联合注入一般是用group_concat(s1,s2,…,sn)函数，其将一个字段的所有函数一次性返回。

4、mid(s,n,len)\substr(s,n,len)\substring(s,n,len)：从s中的n位置开始截取长度为len的子字符串

一般用于盲注。

5、right(s,n)：返回字符串s的后n个字符

一般用于盲注。

6、left(s,n):返回字符串s的前n个字符

一般用于盲注。

7、if(expr,v1,v2)：如果表达式 expr 成立，返回结果 v1；否则，返回结果 v2

一般结合sleep(),用于时间盲注。

8、strcmp(s1,s2)：比较s1和s2，如相同返回0，s1>s2返回1，否则返回-1

一般用于盲注,这里的payload比较微妙。

当s1与s2相同时会返回0，这时候在if(expr,v1,v2)中会进入v2，从而构成判断逻辑。

mysql的udf

介绍：

mysql(user defined function)用户自定义函数，是mysql的一个拓展接口。

用户可以通过自定义函数拓展mysql的功能。

其添加的新函数都可以在sql语句中调用。

原理：

利用了数据库的root高权限，创建系统函数的udf动态函数库，从而就可以利用系统权限进行提权操作。
