Rj45's blog
2019-07-26

安装

运行

参数解释：
-d是于后台运行
-P是将容器内部使用的网络端口映射到主机上
-p是自定义映射端口

绑定网络地址和UDP端口：
docker run -d -p 127.0.0.1:5000:5000/udp training/webapp python app.py

–name可以指定容器名
