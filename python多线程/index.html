1、主线程和子线程

1)启动一个线程就是把一个函数传入并创建Thread实例，然后调用start()开始执行，调用join()表示其结束。

2)在demo:

子线程的名字，在创建时可以指定，如果不指定则线程命名位Thread-1，Thread-2…

3)current_thread()
对于主线程和子线程，任何进程都会默认启动一个线程，这叫主线程。主线程启动的线程叫子线程。
current_thread()函数返回当前线程的实例，current_thread().name，则返回实例的名字。
